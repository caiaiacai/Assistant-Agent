# SATH 用户手册

> **软件名称**：SATH — 感知型智能待办中枢
> **版本**：V1.0

---

## 1. 简介

SATH 是一款桌面级智能待办中枢，用户只需"敲一句话"，软件会自动识别意图、分类、规划，并调度多 Agent 流水线完成调研、准备、提醒等全流程工作。

### 1.1 主要功能
- 一键捕获（全局快捷键 `⌘⇧Space`）
- 智能意图分类：调研 / 任务 / 笔记 / 提醒
- 多 Agent 自动执行调研与准备
- 自然语言时间解析（"下午 3 点"、"10 分钟后"）
- 到期系统通知
- 系统原生定位（CoreLocation）
- 微信消息接入
- Obsidian 笔记双向同步

## 2. 安装与首次启动

### 2.1 系统要求
- macOS 11 Big Sur 或更新版本
- 至少 200 MB 可用磁盘空间
- 联网环境（用于 LLM 和网络搜索）

### 2.2 安装步骤
1. 下载 `sath_x.x.x_aarch64.dmg`（或对应架构）安装包；
2. 双击打开，将 `sath.app` 拖入 "应用程序" 文件夹；
3. 首次打开时，若提示"无法打开，来自未识别的开发者"，请前往 **系统设置 → 隐私与安全性**，点击"仍要打开"。

### 2.3 首次启动引导
启动后进入 **Onboarding 引导页**，分三步：
1. **身份画像**：填写你的角色、当前关注、典型工作、常用工具、偏好；
2. **LLM 配置**：填入你的 LLM 服务商 base_url、api_key、model 名称（支持 OpenAI 兼容协议）；
3. **完成**：进入主界面。

**提示**：Persona 信息越详细，Planner 给出的分类和计划越贴合你的实际工作。

## 3. 主界面操作

### 3.1 捕获念头
- **方式一**：主窗口顶部的输入框，敲入内容 → 回车保存；
- **方式二**：任意前台应用中按 `⌘⇧Space` 呼出主窗口，直接输入；
- **方式三**：从微信、Obsidian 同步进来（需先在设置中配置）。

输入后，该 TODO 会立即入库并显示"分类中…"徽章，Planner 在后台完成分类和计划生成（通常 1-5 秒）。

### 3.2 过滤与搜索
过滤栏提供以下选项：
- 全部 / 未完成 / 已完成
- 调研 / 任务 / 笔记 / 提醒

搜索框支持对 TODO 标题、内容、标签、Agent 产出进行模糊搜索。

### 3.3 TODO 操作

每条 TODO 支持以下交互：
- **完成/取消完成**：点击左侧圆形复选框；
- **展开查看 Agent 产出**：点击 TODO 主区域；
- **跑 Agent**：手动触发（仅 research / task 类型显示）；
- **重分类**：重新调用 Planner；
- **删除**：点击右侧"删除"按钮。

### 3.4 Agent 状态指示
TODO 上会显示小宝石状态图标：
- 蓝色 "…"：Agent 运行中
- 绿色 "✓"：Agent 完成
- 红色 "!"：Agent 失败
- 黄色 "⏱"：Agent 超时

展开 TODO 可查看 Agent 产出的完整 Markdown 报告（支持 GFM 扩展、代码高亮、表格）。

## 4. 设置页

点击顶栏右侧"设置"按钮进入。

### 4.1 用户身份画像（Persona）
LLM 理解你"是谁"的基础。填写越精确，分类越准。

### 4.2 LLM 配置
- **Base URL**：如 `https://api.openai.com/v1`、`https://api.deepseek.com`
- **API Key**：你的密钥
- **Model**：模型名，如 `gpt-4o-mini`、`deepseek-chat`
- **测试连接**：验证配置是否可用

### 4.3 Agent 配置
- **启用 Agent**：总开关
- **搜索提供商**：Tavily / Brave / 无
- **搜索 API Key**：选 Tavily/Brave 时必填
- **最大轮次**（max_turns）：单个 Agent 的最大 LLM 调用次数上限，默认 6
- **最大 Token**：单次 LLM 调用的 max_tokens，默认 2000
- **超时秒数**（timeout_secs）：单轮超时
- **高德 Web Service Key**（可选）：用于反查城市/地址

### 4.4 风险等级
- **保守（Conservative）**：只允许只读网络工具（`web_search`、`web_fetch`、`fs_read`、`fs_list`），最安全
- **标准（Standard）**：+ 沙箱文件写入（`fs_write`、`file_download`）
- **大胆（Bold）**：+ 系统 Shell（`shell_run`），风险最高

### 4.5 情境采集（Context）
- **启用**：允许 SATH 采集最近的窗口 / 浏览器 / 粘贴板片段给 Planner 做上下文参考；
- **窗口大小**（window_minutes）：最近多少分钟的片段；
- **预览**：查看当前即将发给 LLM 的情境摘要。

### 4.6 当前位置
- **自动定位**：应用启动时会通过 macOS CoreLocation 请求一次定位；
- **刷新位置**：手动重新定位；
- **权限**：首次运行时会弹出系统定位权限对话框，选"允许"即可。若拒绝，可前往 **系统设置 → 隐私与安全性 → 定位服务** 手动开启。

### 4.7 Obsidian 集成
- **启用**
- **Vault 路径**：你的 Obsidian vault 绝对路径
- **子文件夹**：SATH 在 vault 内的写入目录
- **测试连接** / **全量同步**

### 4.8 微信集成
- **启用**
- **登录**：弹出二维码，用手机微信扫码登录；
- **退出登录**：清除本地 session

启用后，任何发给"微信文件传输助手"或你指定联系人的消息会被当作 TODO 写入 SATH；Agent 完成后会把关键信息用微信消息回发。

## 5. 常用场景示例

### 5.1 定时提醒
**输入**："下午 3 点开会"
**结果**：
- 分类为 `reminder`
- 本地正则解析时间 → 自动填入 `due_at` 为今天 15:00
- 到期时弹出系统通知

### 5.2 带准备的提醒
**输入**："明天下午 3 点打篮球"
**结果**：
- 分类为 `reminder + research`
- Planner 启动调研 Agent：查天气、附近球场、是否下雨
- 到期通知 + 打开 TODO 可看到完整调研报告

### 5.3 主动调研
**输入**："调研一下 Cursor 和 Windsurf 的差异"
**结果**：
- 分类为 `research`
- Pipeline 可能包含"竞品分析"和"文档汇总"两个 Agent，并行执行
- 产出 Markdown 报告，含功能对比表、价格、用户评价等

### 5.4 普通笔记
**输入**："今天和王总聊了下，他们对我们的方案很感兴趣"
**结果**：
- 分类为 `note`
- 无 Agent 执行，纯记录

## 8. 常见问题

### Q1：定位始终显示"未知城市"？
- 确认 **系统设置 → 隐私与安全性 → 定位服务** 已开启且 SATH 已被勾选；
- 确认已在"设置 → 当前位置"里点过"刷新位置"；
- 若要显示中文城市名，需要在 **设置 → Agent 配置** 里填入高德 Web Service Key。

### Q2：Agent 总是跑失败或超时？
- 打开设置，把 **最大轮次** 调高到 8-10；
- 把 **超时秒数** 调高到 600；
- 检查 LLM 是否能正常响应（点"测试连接"）；
- 检查搜索 API Key 是否正确。

### Q3：微信扫码后没反应？
- 检查是否开启微信网关；
- 退出登录重新扫码；
- 查看终端日志是否有错误。

### Q4：下午 3 点的提醒时间显示成去年？
- 已修复。新版本 Planner prompt 会注入当前日期，且本地 parser 结果会覆盖 LLM 输出。

### Q5：TODO 列表滚动卡？
- 当前每 2.5 秒轮询一次，后续会改成事件驱动增量更新。
