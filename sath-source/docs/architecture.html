Design Principles
01
影子模式
一切 Agent 后台运行，绝不抢焦点、不弹窗、不打断
02
秒级创建

隐私本地

乐观锁
Agent 回填不覆盖用户修改，version 冲突自动 merge
05
读写隔离
调研类并发执行，写操作串行，来自 CC 源码的设计
06
极简界面
列表 + 单行输入，没有聊天框，Dieter Rams 风格

Smart TODO Object · Data Schema
todo
├─ id TEXT PK // hex(randomblob(8))
├─ title TEXT // 任务标题 ≤30字
├─ body TEXT // Markdown 富文本
├─ status ENUM // inbox → active → waiting → done → archived
├─ intent ENUM // research | task | record | reminder
├─ priority INT 0-4 // 0=none → 4=urgent
├─ confidence REAL // AI 判定置信度
├─ context_snapshot JSON // 创建时环境快照 (冻结)
├─ enrichment JSON // Agent 异步回填的增强内容
├─ enrichment_status ENUM // none → pending → running → done
├─ sync_state JSON // {apple: {}, lark: {}, notion: {}}
├─ source_tag TEXT // 防循环标记
├─ version INT // 乐观锁
└─ project_id FK // → project.id

Architecture Inspirations

← Claude Code 源码

  • AsyncLocalStorage 隔离并发 Agent
  • 读写分离: 只读并发, 写操作串行
  • 分层 Context: System + User + Tool
  • Hook 系统: Pre/Post Tool 生命周期
  • 乐观锁 + version 冲突检测
  • MessageQueue 优先级调度

← Hermes Agent

  • Skill 注册表: Agent 按类型注册
  • 持久化记忆: 跨会话状态
  • Subagent 隔离: 并行工作流
  • Gateway 抽象: 多平台统一入口
  • Cron 调度: 定时任务内置
  • RPC 工具调用协议
